## Dev Commit format

-   :new: Features/Improvements.
-   :up: General Upgrade.
-   :arrow_up: Upgrade Version.
-   :package: Package.
-   :bug: Bug/Error.
-   :fire: Delete/Clean.
-   :memo: Comments/Docs.
-   :triangular_flag_on_post: Red Flag.
-   :rewind: Rollback.
